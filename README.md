# InstantShop

InstantShop turns Instagram Business posts into sellable e-commerce products using AI. Connect Instagram via Facebook OAuth, let AI analyze your posts into products, and manage everything through the admin dashboard with a public storefront.

## Development

```bash
npm install
npm run dev      # Vite dev server on port 5173
npm run build    # Production build → dist/
npm run lint     # ESLint
```

See `CLAUDE.md` and `DOCUMENTATION.md` for architecture details.
